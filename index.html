Google Sheets Setup

Publish your Google Sheet as a CSV and paste the URL below. Your sheet must have these column headers in row 1:

Agreement ID Property Unit Tenant Name Start Date Expiry Date Rent Escalation Date Escalation % Contact Email Status

File → Share → Publish to web → Select "Comma-separated values (.csv)" → Publish → Copy link. Or use the direct export URL.

Email notifications require a backend (e.g. EmailJS, SMTP relay). This address is saved for reference and mailto: links.

How to use:
1. Create a Google Sheet with the column headers listed above.
2. Add your lease data row by row. Dates should be in YYYY-MM-DD or DD/MM/YYYY format.
3. Publish the sheet as CSV (File → Share → Publish to web).
4. Paste the CSV URL in settings above and click Save Settings.
5. Click "Sync Data Now" — your leases will appear on the dashboard.
6. Enable push notifications for browser alerts 60 and 30 days before expiry/escalation.
